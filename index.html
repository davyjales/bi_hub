Abrir relatórios sem a Microsoft Store

O navegador costuma tratar links file:// para .pbix de forma errada. Para evitar que abra a Microsoft Store, instale o assistente:

Clique aqui para baixar o BI-Hub-Helper.ps1

O arquivo deve cair em C:\Users\\Downloads\BI-Hub-Helper.ps1. Em seguida, execute-o (botão direito → Executar com PowerShell) e deixe a janela aberta; ao abrir um BI, só uma nova guia mostra a confirmação — o hub permanece nesta.

Se “executar com botão direito” não funcionar, execute no CMD/PowerShell como Administrador: powershell.exe -ExecutionPolicy Bypass -File "C:\\Users\\\\Downloads\\BI-Hub-Helper.ps1"
